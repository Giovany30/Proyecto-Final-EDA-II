
Para compilar en Windows:

$ gcc -Wall -std=c99 -fopenmp List.h List.c Queue.c Queue.h Graph.c Graph.h Refugio.c Refugio.h cJSON.c main.c -o main.exe

Para compilar en Linux:

$ gcc -Wall -std=c99 -fopenmp List.h List.c Queue.c Queue.h Graph.c Graph.h Refugio.c Refugio.h cJSON.c main.c -o main.out

(Recuerda que el símbolo $ no debe ser introducido, éste sólo indica el prompt de la terminal.)
